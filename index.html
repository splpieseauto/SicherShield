Cloud Sync preia automat fișierele din Drive și le salvează pe NAS în câteva secunde.
🖥️ Configurare Cloud Sync
1
Instalează Cloud Sync
DSM → Package Center → caută „Cloud Sync" → instalează gratuit
2
Conectează Google Drive
Cloud Sync → + Adaugă → Google Drive → autorizează cu același cont Google
3
Configurează task sincronizare
Folder Cloud Drive: /SicherShield
Folder NAS local: /volume1/SicherShield
Direcție: unidirecțional Download (Drive → NAS)
4
Activează sincronizarea
Apasă Start — de acum orice fișier încărcat în Drive apare automat pe NAS în 10–60 secunde
⚙️ Preferințe sincronizare
✅ Recomandat: Sync unidirecțional Drive → NAS. Drive = sursă de adevăr, NAS = arhivă locală redundantă. Nu edita fișierele direct pe NAS.

🐙 Publicare pe GitHub Pages

GitHub Pages găzduiește SicherShield gratuit pe HTTPS — necesar pentru NFC, Google Drive API și fonturi. Fișierul HTML se încarcă direct, fără build tools.

1
Creează cont și repository GitHub
Mergi la github.com → New repository
Nume: sichershield · Visibility: Public
2
Încarcă fișierul SicherShield.html
În repository → Add file → Upload files
Redenumește fișierul în index.html înainte de upload
3
Activează GitHub Pages
Repository → Settings → Pages
Source: Deploy from a branch · Branch: main · Folder: /(root)
4
URL-ul tău HTTPS
După ~2 minute aplicația e disponibilă la:
https://username.github.io/sichershield/
Adaugă acest URL în Google Cloud Console → Authorized JS origins (pentru Drive API)
5
Update fișier (pentru versiuni noi)
Repository → click pe index.html → ✏️ Edit (creion) → sau
Add file → Upload files → suprascrie fișierul → Commit changes
GitHub Pages se actualizează automat în ~60 secunde
